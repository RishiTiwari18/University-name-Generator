# University Finder 

University Finder is a simple web application that allows users to enter the name of **any country** and get a list of **all universities available in that country**.

This project uses a public API to fetch real university data and displays it dynamically on the web page.

# Features

- Search universities by country name  
- Displays a list of universities instantly  
- Simple and clean user interface  
- Uses real-time data from an external API  
- Beginner-friendly JavaScript project  

# Technologies Used

- HTML  
- CSS  
- JavaScript  
- API (Universities API)  

# How It Works

1. User enters a **country name** in the input field  
2. Clicks the **Search** button  
3. The app sends a request to the API  
4. All universities related to that country are displayed
